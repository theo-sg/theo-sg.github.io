YMIR

A programming language and Python interpreter

View the source code here

During my first year of GCSE Computer Science, I began learning about compilers and how they produced machine code from a high-level language. Over the summer, during my 3-day placement at HSBC, I developed a very simple Python interpreter that is able to execute a program line-by-line. I based this programming language heavily on Assembly, as it works using simple mnemonics as well as directly addressing memory locations. Despite its simplicity, I was able to create complex programs such as a random number guessing game!

This was also my first introduction to using Git and version control for a project, a skill that has become invaluable over the last few years. Evaluating this project also shows me how far I have come as a programmer in the past years, as I wouldn't dream of doing some of the things I did in this project now!
