GUILDS PLUGIN

A Minecraft plugin that allows players to make their own factions

View the source code here

At the beginning of the first COVID-19 lockdown in March 2020, I set up and hosted a Minecraft server for myself and my friends to play on. I installed Ubuntu on an old PC and installed it as a NAS drive and server host on my local network, as well as setting up port-forwarding rules and even a dynamic DNS system to ensure that connection was easy.

In this time, I created a variety of utility programs, such as a remote batch file opener and NAS syncing utility, and controlled the operation remotely via SSH from my main Windows PC. I also began experimenting with Minecraft plugins using the Spigot API. The main one I created was a complete teams system, which allowed players to create, customise, and manage their own teams in-game. This system was a great addition to the game and my friends were very impressed with how in-depth I had gone with the plugin.

As my first project that had to be rigorously tested due to actual user requirements, as well as my first complete project in Java, I think this project turned out very well. I used comprehensive unit tests to ensure everything went smoothly.
