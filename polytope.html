POLYTOPE

A work-in-progress action platformer

View the source code here

POLYTOPE is a 2D action puzzle platformer game, currently in development.

I started work on the game initially as a game jam idea back in 2020, developing a very basic prototype for a physics-based platformer which incorporated both 2D and 3D elements. When lockdown began in 2021, I decided to see if I could develop this idea further into a more fleshed out game prototype. For this project, my main goal was to actually finish and release, however with all of the events of 2021 unfortunately I couldn't find the time!

I would consider this to be one of my most impressive projects so far, as most of my code is very functional, well arranged and well commented, implementing features such as:

  • Individual manager scripts using singletons
  • Layer-based collision detection and calculations for physics interactions
  • Raycasting for wall-jump mechanics
  • Gravity and drag emulation on 2D rigidbodies
  • Event-based input systems (instead of Unity's regular checking for input each frame)

I have lots of ideas for this project that I will be adding in the future, so stay tuned!
